# 安全说明

## 安装或运行前

第三方 Skill、工作流和脚本可能读取或修改本地文件、访问网络、调用外部 API，或者要求配置账号权限。请先完成以下检查：

1. 阅读完整的 `SKILL.md`、脚本和引用文件。
2. 确认来源、提交版本和许可证。
3. 搜索 API Key、Token、Cookie、上传、删除、覆盖和外部命令相关操作。
4. 只授予完成任务所需的最小权限。
5. 对 Excel、PPT、Word、Excalidraw 等文件先复制备份，再在副本上测试。
6. 不要把真实客户数据、公司机密或个人隐私直接交给未经核验的第三方工具。

## 已知风险提示

本次第三方快照中的 Excalidraw 辅助脚本包含重命名、删除临时文件和覆盖目标文件的逻辑。静态扫描未发现脚本主动联网或读取环境密钥，但静态扫描不能替代沙箱测试和人工审查。

部分 Markdown 文档提到 API Key、外部发布平台、ATS、YouTube 下载工具或其他第三方服务。这些是第三方工作流说明，不代表本仓库建议启用相应服务或授予权限。

## 密钥处理

- 不要把真实密钥写进 Markdown、Skill、Git 提交或截图。
- 使用系统环境变量或受控的密钥管理方式。
- 如果密钥意外提交，立即撤销并更换；仅从 Git 历史删除并不能使已泄露的密钥重新安全。

## 报告问题

发现敏感信息、恶意行为、许可证问题或高风险脚本时，请通过 GitHub Issue 提交最小复现信息，不要在 Issue 中粘贴真实密钥、个人数据或公司机密。
