## 🚀 快速开始

### 安装

```bash
# 克隆本仓库
git clone https://github.com/brizer/awesome-niuma-skills.git

# 进入目录
cd awesome-niuma-skills

# 查看所有技能
ls skill/
```

### 选择你的战场

```
skill/
├── HR人力资源/          # 39个技能 ── 求职简历不求人
├── 产品经理/            # 56个技能 ── 需求文档秒完成
├── 图形处理/            # 5个技能  ── 图表动效一键造
├── 新媒体运营/          # 63个技能 ── 流量密码我全要
├── 电商运营/            # 7个技能  ── 数据驱动GMV
├── 程序员_开发/          # 6个技能  ── 代码优雅不秃头
└── 行政文员/            # 5个技能  ── 杂活也能干漂亮
```

### 一分钟上手

```bash
# 假设你想优化简历
cd skill/HR人力资源/简历优化
cat README.md  # 查看技能详情

# 假设你是产品经理在写PRD
cd skill/产品经理/需求与文档
cat README.md  # 开工！
```

## 🎓 如何使用

### 方法一：直接查阅

```bash
# 找到你需要的Skill分类
cd skill/你想要的分类

# 查看该Skill的详细说明
cat README.md
```

### 方法二：按需取用

每个Skill都是一个独立的工作流/模板/指南。你可以：

1. **直接使用**：复制粘贴到你的工作中
2. **修改定制**：根据你的实际情况调整
3. **组合创造**：多个Skill叠加使用效果更佳

### 方法三：贡献新Skill

```bash
# 1. Fork 本仓库
# 2. 创建你的Skill目录
mkdir -p skill/你的分类/你的Skill名称

# 3. 编写Skill内容
cat > skill/你的分类/你的Skill名称/README.md << 'EOF'
# 你的Skill标题

> 一句话描述

## 适用场景
...

## 使用方法
...
EOF

# 4. 提交PR，等待合并
```
